"RestroPress - Address Auto Complete" enhances the ordering process by suggesting addresses as users type, improving accuracy and efficiency.

== Description ==

This plugin is very useful for users to fill in the order details forms with address autocomplete functionality, so they will not need to provide the details for every input field one by one.

This plugin uses Google Maps API integration, so to use this plugin you must have a valid Google Maps API Key.

**Third-Party Service Usage:**  
This plugin relies on the Google Maps API service to provide address autocomplete functionality. By using this plugin, you acknowledge and agree to the terms of use and privacy policy of the Google Maps API service.

* [Google Maps API](https://developers.google.com/maps/documentation/javascript/get-api-key)
* [Google Maps API Terms of Use](https://developers.google.com/maps/terms)
* [Google Privacy & Terms](https://policies.google.com/privacy)

== Installation ==

This section describes how to install the plugin and get it working.

1. Log in to your WordPress dashboard and navigate to Plugins > Add New.
2. Search for "RestroPress - Address Auto Complete".
3. Click install.
4. Click activate.
5. Once the plugin is installed, you can see RestroPress on the left navigation bar of the WordPress Dashboard.

== Frequently Asked Questions ==

= What is RestroPress and how to download RestroPress? =

RestroPress is an online food ordering system for WordPress. It is a standalone WordPress plugin that allows you to easily add a food ordering system to your WordPress website. Using RestroPress, you can easily receive both Pickup/Takeaway and Delivery orders.

RestroPress has a user-friendly frontend and backend interface that will allow you to easily manage orders and also comes with a pre-built user dashboard to manage their profile and orders.

The plugin automatically adds the required pages to your site when you activate it. The plugin displays the food items on your page using the [fooditems] shortcode.

[Download RestroPress](https://restropress.com/checkout?edd_action=add_to_cart&download_id=53444)
